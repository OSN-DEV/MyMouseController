# MyMouseController
ホットキーをつかってマウスカーソルをアクティブウィンドウ内に移動する、だけのツールです。

# 使い方
* Ctrl + Shift + Z ： アクティブウィンドウの中央にカーソルを移動(既にカーソルがアクティブウィンドウ内にあるときは何もせず)
* Ctrl + Shift + X ： アクティブウィンドウの右上(閉じるボタン付近)にカーソルを移動
* Ctrl + Alt + ← → ： 左右のモニタを移動
* Ctrl + ALt + U :  画面左上
* Ctrl + ALt + J :  画面左中央
* Ctrl + ALt + M :  画面左下
* Ctrl + ALt + I :  画面中央上
* Ctrl + ALt + K :  画面中央
* Ctrl + ALt + , :  画面中央下
* Ctrl + ALt + O :  画面右上
* Ctrl + ALt + L :  画面右中央
* Ctrl + ALt + . :  画面右下
